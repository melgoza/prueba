Funcionamiento de etiquetas en HTML

En programación existen etiquetas para indicar acciones, una de estas es "section", esta etiqueta sirve para separar en partes en un archivo, como su nombre lo dice, secciona los archivos. También les sirve a los programadores a identificar mas fácil las diferentes partes de un documento.

El elemento "div" es otra etiqueta. Esta etiqueta sirve para dividir en partes una parte de un documento, esta etiqueta generalmente se usa dentro de una etiqueta "section", section secciona el documento y div divide en partes esas secciones de los documentos.

La etiqueta main sirve para contener la información principal de un documento. Esta etiqueta se pone a dentro de "body" (el cuerpo del documento).

La función de la etiqueta article es separar la informacion de un documento, como párrafos o ideas, generalmente se usa para separar los artículos de un documento.

Aside es una etiqueta en la que se pone la información que va a un lado de la página. Sirve para poner información secundaria sobre el tema, o también muchas veces se usa para poner anuncios.
